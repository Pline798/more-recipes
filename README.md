# 更多配方 (More Recipes)

一个 **Fabric** 模组，为 Minecraft 1.21.11 添加矿物互转、磁铁、实用配方与自定义物品。

---

## 📦 内容

### ✨ 自定义物品

| 物品 | 说明 |
|------|------|
| 🧪 **大附魔之瓶** | 投掷后爆出 **30~110 XP**（原版 10 倍），附魔光泽，可堆叠 64 个 |
| 🪨 **石炭** | 熔炉燃料 **30000 ticks**（原版煤炭 30 倍），由黑曜石烧制 |
| ⚡ **转换核心** | 矿物向上转换的催化剂，合成后自动返还 |
| 🔄 **逆转核心** | 矿物向下转换的催化剂，合成后自动返还 |
| 🧲 **磁铁** | 放在背包中自动吸取附近掉落物与经验球，右键切换开关。共 4 级：铁（8 格）、金（12 格）、钻石（16 格）、下界合金（20 格） |

### 📖 附魔：速攻

| 属性 | 值 |
|------|:---|
| 效果 | 攻击速度 **+50%/级**（`add_multiplied_base` × 0.5） |
| 适用 | 剑、斧等锋利类武器 |
| 等级 | I–III |
| 权重 | 5 |
| 获取 | 图书管理员村民交易（已加入 `#minecraft:enchantment/tradeable`） |

### 🔥 熔炉烧制

| 输入 | 产出 | 经验 | 时间 |
|------|------|:----:|:----:|
| 腐肉 → | 皮革 | 0.1 XP | 200 ticks (10s) |
| 青金石 → | 经验瓶 | 0.3 XP | 200 ticks (10s) |
| 钻石 → | **大附魔之瓶** | 0.7 XP | 200 ticks (10s) |
| 黑曜石 → | **石炭** | 0.5 XP | 600 ticks (30s) |

### 🛠️ 基础合成

| 输入 | 产出 | 方式 |
|------|------|:----:|
| 9× 经验瓶 | **大附魔之瓶** | 3×3 满格填充 |
| **大附魔之瓶** | 9× 经验瓶 | 无序拆解 |
| 黑曜石 + 水桶 | **哭泣的黑曜石**（水桶返还） | 无序 |
| **石炭** + 木棍 | **火把 ×16** | 无序 |
| 钻石 ×4 + 红石 | **转换核心** | 十字型 |
| 红石 ×4 + 钻石 | **逆转核心** | 十字型 |

#### 🧲 磁铁合成

| 产物 | 范围 | 合成方式 |
|:-----|:----:|:----------|
| **铁磁铁** | 8 格 | 铁锭 ×8 围红石 |
| **金磁铁** | 12 格 | 金锭 ×8 围铁磁铁 |
| **钻石磁铁** | 16 格 | 钻石 ×8 围金磁铁 |
| **下界合金磁铁** | 20 格 | 锻造台：下界合金升级模板 + 钻石磁铁 + 下界合金锭 |

### ⚡ 矿物互转

使用 **转换核心** 向上转换矿物，**逆转核心** 向下转换矿物。核心在合成后自动返还（不消耗）。

| 向上（转换核心） | 向下（逆转核心） |
|:-----------------|:-----------------|
| 煤炭 ×2 → 铜锭 | 铜锭 → 煤炭 ×2 |
| 煤炭 ×4 → 铁锭 | 铁锭 → 煤炭 ×4 |
| 煤炭 ×16 → 金锭 | 铁锭 → 铜锭 ×2 |
| 铜锭 ×2 → 铁锭 | 金锭 → 铁锭 ×2 |
| 铜锭 ×4 → 金锭 | 金锭 → 铜锭 ×4 |
| 铁锭 ×2 → 金锭 | 钻石 → 金锭 ×2 |
| 金锭 ×2 → 钻石 | 青金石 → 红石 ×2 |
| 红石 ×2 → 青金石 | |

### 🎯 掉落物

击杀以下生物时有几率掉落 0–2 个皮革（通过 `LootTableEvents.MODIFY` 追加，不覆写原版）：

猪 · 羊 · 猪灵 · 僵尸猪灵 · 疣猪兽

### 🏆 成就

| 成就 | 触发条件 |
|------|:---------|
| **转换公式** | 获得转换核心 |
| **逆转公式** | 获得逆转核心 |

## 🔧 安装

1. 安装 **Fabric Loader** ≥ 0.19.2
2. 安装 **Fabric API**（≥ 0.141.4）
3. 将 `more-recipes-<version>.jar` 放入 `.minecraft/mods/`

### 依赖

| 依赖 | 版本 |
|------|:----:|
| Minecraft | 1.21.11 |
| Fabric Loader | ≥ 0.19.2 |
| Fabric API | ≥ 0.141.4+1.21.11 |
| Java | ≥ 21 |

---

## 🏗️ 自行构建

```bash
git clone https://github.com/Pline798/more-recipes.git
cd more-recipes
./gradlew build
```

产物位于 `build/libs/more-recipes-<version>.jar`

## 📁 项目结构

```
src/
├── main/java/com/para/morerecipes/
│   ├── MoreRecipes.java                   # 主入口：注册物品/实体/附魔/创造标签页
│   ├── component/
│   │   └── ModComponents.java             # 自定义数据组件（磁铁开关状态）
│   ├── entity/
│   │   └── ThrownBigExperienceBottle.java # 大附魔之瓶实体（10倍经验）
│   ├── item/
│   │   ├── BigExperienceBottleItem.java   # 大附魔之瓶物品行为
│   │   └── MagnetItem.java               # 磁铁物品行为（吸引+右键开关）
│   └── mixin/accessor/
│       └── ItemAccessor.java              # Mixin 访问器（设置合成残留物）
├── client/java/com/para/morerecipes/
│   └── client/
│       └── MoreRecipesClient.java         # 客户端入口（注册实体渲染）
└── main/resources/
    ├── fabric.mod.json
    ├── more-recipes.mixins.json
    ├── assets/           # 纹理、模型、语言文件
    └── data/             # 配方、进度、附魔、标签
```
