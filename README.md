# 更多配方 (More Recipes)

一个 **Fabric** 模组，为 Minecraft 1.21.11 添加一系列实用配方与自定义物品。

---

## 📦 内容一览

### 🔥 熔炉烧制

| 输入 | 产出 | 经验 | 烧制时间 |
|------|------|:----:|:--------:|
| 腐肉 | 皮革 | 0.1 XP | 200 ticks |
| 青金石 | 经验瓶 | 0.3 XP | 200 ticks |
| 钻石 | **大附魔之瓶** | 0.7 XP | 200 ticks |
| 黑曜石 | **石炭** | 0.5 XP | 600 ticks |

### 🛠️ 合成台

| 输入 | 产出 | 方式 |
|------|------|:----:|
| 9 经验瓶 | **大附魔之瓶** | 3×3 满格 |
| 1 大附魔之瓶 | 9 经验瓶 | 无序拆解 |
| 黑曜石 + 水桶 | **哭泣的黑曜石** | 无序 |
| 石炭 + 木棍 | **火把 ×16** | 无序 |

---

## ✨ 自定义物品

### 🧪 大附魔之瓶
- 右击投掷，爆出 **30~110 XP**（原版经验瓶 10 倍）
- 附魔光泽
- 模型比原版大 **20%**

### 🪨 石炭
- 黑曜石熔炉烧制获得
- **6400 ticks** 燃料（原版煤炭 4 倍，烧 320 个物品）
- 1 石炭 + 1 木棍 → **火把 ×16**

---

## 📖 附魔

### ⚔️ 速攻 (Fast Attack)
- **攻击速度 ×2** — 攻击间隔减半
- 适用剑、斧等武器
- 等级 I（单级）
- 获取：图书管理员村民有几率出售

---

## 🔧 安装

1. 安装 **Fabric Loader** ≥ 0.19.2
2. 安装 **Fabric API**
3. 将 `more-recipes-1.0.0.jar` 放入 `.minecraft/mods/`

### 依赖

| 依赖 | 版本 |
|------|:----:|
| Minecraft | 1.21.11 |
| Fabric Loader | ≥ 0.19.2 |
| Fabric API | * |
| Java | ≥ 21 |

---

## 🏗️ 自行构建

```bash
git clone https://github.com/Pline798/more-recipes.git
cd more-recipes
./gradlew build
```

产物：`build/libs/more-recipes-1.0.0.jar`
