# 更多配方 (More Recipes)

一个 **Fabric** 模组，为 Minecraft 1.21.11 添加矿物互转、实用配方与自定义物品。

---

## 📦 内容

### 🔥 熔炉烧制

| 输入 | 产出 | 经验 | 时间 |
|------|------|:----:|:----:|
| 腐肉 | 皮革 | 0.1 XP | 200 ticks |
| 青金石 | 经验瓶 | 0.3 XP | 200 ticks |
| 钻石 | **大附魔之瓶** | 0.7 XP | 200 ticks |
| 黑曜石 | **石炭** | 0.5 XP | 600 ticks |

### 🛠️ 通用合成

| 输入 | 产出 | 方式 |
|------|------|:----:|
| 9 经验瓶 | **大附魔之瓶** | 3×3 满格 |
| 1 大附魔之瓶 | 9 经验瓶 | 无序拆解 |
| 黑曜石 + 水桶 | **哭泣的黑曜石** | 无序 |
| 石炭 + 木棍 | **火把 ×16** | 无序 |
| 钻石 ×4 + 红石 | **转换核心** | 十字型 |
| 红石 ×4 + 钻石 | **逆转核心** | 十字型 |

### ⚡ 矿物互转（需转换核心/逆转核心）

核心在合成后自动返还。

| 向上（⚡转换核心） | 向下（🔄逆转核心） |
|:-------------------|:-------------------|
| 煤炭×2 + ⚡ → 铜锭 | 铜锭 + 🔄 → 煤炭×2 |
| 煤炭×4 + ⚡ → 铁锭 | 铁锭 + 🔄 → 煤炭×4 |
| 煤炭×8 + ⚡ → 金锭 | 铁锭 + 🔄 → 铜锭×2 |
| 铜锭×2 + ⚡ → 铁锭 | 金锭 + 🔄 → 铁锭×2 |
| 铜锭×4 + ⚡ → 金锭 | 金锭 + 🔄 → 铜锭×4 |
| 铁锭×2 + ⚡ → 金锭 | 钻石 + 🔄 → 金锭×2 |
| 金锭×2 + ⚡ → 钻石 | |

### ✨ 自定义物品

| 物品 | 说明 |
|------|------|
| 🧪 **大附魔之瓶** | 右击投掷，爆出 30~110 XP（原版 10 倍），附魔光泽 |
| 🪨 **石炭** | 黑曜石烧制，6400 ticks 燃料（原版煤炭 4 倍） |
| ⚡ **转换核心** | 矿物向上转换的催化剂，合成后返还 |
| 🔄 **逆转核心** | 矿物向下转换的催化剂，合成后返还 |

### 📖 附魔：速攻

- 攻击速度 ×2（攻击间隔减半）
- 适用剑、斧等武器
- 等级 I
- 获取：图书管理员村民交易

### 🏆 成就

| 成就 | 触发条件 |
|------|:---------|
| **转换公式** | 获得转换核心 |
| **逆转公式** | 获得逆转核心 |

---

## 🔧 安装

1. 安装 **Fabric Loader** ≥ 0.19.2
2. 安装 **Fabric API**
3. 将 `more-recipes-<version>.jar` 放入 `.minecraft/mods/`

### 依赖

| 依赖 | 版本 |
|------|:----:|
| Minecraft | 1.21.11 |
| Fabric Loader | ≥ 0.19.2 |
| Fabric API | * |
| Java | ≥ 21 |

---

## 🏗️ 自行构建

```bash
git clone https://github.com/Pline798/more-recipes.git
cd more-recipes
./gradlew build
```

产物：`build/libs/more-recipes-<version>.jar`
